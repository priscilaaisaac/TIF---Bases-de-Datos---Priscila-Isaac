### Sistema de Gestión Hotelera | Proyecto Integrador – Bases de Datos

Desarrollo de un sistema de gestión para el sector hotelero mediante el diseño e implementación de una base de datos relacional en PostgreSQL. El proyecto abarcó todas las etapas del proceso, desde el modelado conceptual hasta la creación de una interfaz gráfica para la visualización de información.

**Principales actividades desarrolladas:**

* Análisis de los requerimientos del sistema y diseño del modelo de datos.
* Elaboración del Diagrama Entidad–Relación (DER) utilizando la notación de Chen.
* Implementación de la base de datos en PostgreSQL mediante sentencias DDL, definiendo claves primarias, claves foráneas y restricciones de integridad.
* Generación y carga de un conjunto de datos consistente para simular el funcionamiento de una cadena hotelera.
* Desarrollo de consultas SQL avanzadas empleando JOIN, EXISTS, NOT EXISTS, subconsultas y funciones de agregación.
* Optimización de consultas mediante conceptos de álgebra relacional.
* Desarrollo de una aplicación en Python conectada a PostgreSQL mediante psycopg2 para consultar la base de datos y visualizar indicadores a través de gráficos utilizando pandas y matplotlib, con una interfaz gráfica desarrollada en tkinter.
* Documentación técnica del proyecto y publicación del código fuente en GitHub.
